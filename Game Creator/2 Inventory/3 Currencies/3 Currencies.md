
- 货币
- 为了确定物品的价值，使用了货币的概念
- 每个Currency 拥有一个或者多个Conis，每个金币都对应一个单位的价值
-  Create → Game Creator → Inventory → Currency.
![[Pasted image 20251031141149.png]]

- 以货币不能有小数，如果必须要小数
- 要用两位小数表示一个数值，可以 x 100，然后将小数点向左移动两位

== 
- 有些游戏需要采用多种货币，每种货币代表不同的价值
![[Pasted image 20251031143917.png]]
- 拥有三种货币类型，每种货币的价值不同
	- 铜币价值最低
	- 银币相当于25铜币
	- 金币相当于5枚银币
- 因此推出以下value之间的关系
	- Copper = 1 
	- Silver = 25
	- Gold = 125
![[Pasted image 20251031145512.png]]

- 基于value的运算形式，比如说要给玩家一个金色货币，则其财富增加125就好
