## Merchant Component 

- 商人，即发起交易的人，其Bag组件上需要有一个Merchant组件
	- Bag组件提供物品库存（stock）
	- Merchant组件决定了进行交易的类型
![[Pasted image 20251031152237.png]]

### Merchant Info

- Merchant info, Merchant section 部分允许给与商家命名即添加描述。可以给予某个有一定类型的Mechant一种标签
- 比如某个商人叫草药学家，即这个交易的物品类型就确定了
![[Pasted image 20251031153017.png]]
### Configuration


| **Infinite Currency** | 选中后，将拥有无限的货币存储，可用于从玩家处购买物品（sell），否则只能使用以物换物                              |
| --------------------- | ------------------------------------------------------------------------ |
| **Infinite Stock**    | 无限库存，可用库存数量不会减少，否则不管如何都会减少可用库存                                           |
| **Allow Buy Back**    | 选中后，玩家售出的内容都将到商人的库存中                                                     |
| **Sell Niche Type**   | 选中后，，允许筛选此商人出售的物品类型，如果商人只出售草药，即使背包中包含剑，该剑也不会出售                           |
| **Buy Rate**          | 1表示无折扣，0.9表示90&的折扣，该字段为0.9                                               |
|                       |                                                                          |
| **Sell Rate**         | 值商人向玩家处购买物品时应用的系数，通常，物品的出售价格低于实际价格（通常是实际价格的一般）                           |
| **Bag**               | 指商人从中获取库存的背包组件，如果背包在其他组件上可以引用一下再<br>![[Pasted image 20251031155508.png]] |
| **Skin UI**           | 商家使用的用户界面 ui                                                             |
