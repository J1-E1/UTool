- 将物品转化为其他物品的过程，“改造”
	- Crafting：将多个物品合成为一个
	- **Dismantling**：拆解，摧毁一个item为了回收多个物品


![[Pasted image 20251031202455.png]]
- 此指令使用一个 合成和拆解的ui界面来完成这个操作
- Input Bag 和 output Bag 是改造过程中使用的袋子
- 可能会出现新的袋子
- **Filter Item** 字段决定了item显示的类型，Filter type可能是通过不同的类型来区分其可制造的装备，比如药水或者装备
